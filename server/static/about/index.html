CentOS Container Registry


Welcome to the CentOS Container Registry

The CentOS Container Registry is a centralized hub for making containers easily accessible for all developers.

Have your containers built, analyzed, maintained, and scanned in one centralized registry.

Features

The CentOS Container Registry makes sure that your container is always up-to-date. Built-in analysis tools will alert you for any need-to-know changes. Each time that an updated is detected within the CentOS base image, a new build is triggered.

Submitting containers

It's as simple as opening up a pull request. Follow the instructions on GitHub
